# 🚇 Metro Rail Management & Analytics System

## Project Overview
This project is a MySQL-based Metro Rail Management and Analytics System designed to manage metro operations including passengers, stations, routes, trains, ticketing, payments, employees, smart cards, and maintenance activities.

The project demonstrates database design, SQL programming, performance optimization, and business analytics using real-world metro rail scenarios.

---

## Database Features

- 10 Relational Tables
- Primary Keys & Foreign Keys
- Data Integrity Constraints
- Joins (Inner, Left)
- Subqueries & Correlated Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Views
- Stored Procedures
- Triggers
- Indexing & Query Optimization

---

## Business Questions Solved

- Total passengers registered by city
- Top revenue-generating cities
- Monthly and yearly revenue trends
- Busiest metro stations
- Most profitable routes
- Top frequent travelers
- Smart card usage analysis
- Maintenance cost analysis
- Stations without employees
- Passengers without ticket bookings
